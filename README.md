# 📊 Telecom Churn Prediction

## 📌 Overview
This project focuses on predicting whether a telecom customer will churn using machine learning classification models.

- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building (Random Forest, SVM)
- Model Evaluation

## 🛠️ Tech Stack

- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📈 Results
- Built classification models to predict churn
- Achieved good accuracy and performance
